Apa itu DDL?

DDL (Data Definition Language) adalah ...

Posted by Story of Arlin on September 18, 2019

DDL (Data Definition Language) adalah kumpulan perintah SQL yang digunakan untuk membuat (create), mengubah (alter) dan menghapus (drop) struktur dan definisi tipe data dari objek-objek database.

Objek-objek database yang dimaksud pada MySQL adalah sebagai berikut:

  • Database
  • Table
  • View
  • Index
  • Procedure (Stored Procedure)
  • Function
  • Trigger
